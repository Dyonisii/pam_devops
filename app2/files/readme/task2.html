Task 2 - Log analysis

In this task you have to use Linux command line tools in conjunction, to quickly and efficiently analyze large amount of logs, no coding required. All the necessary tools are already available inside the Docker container, they include but are not limited to: zstdcat, zstdgrep, grep, sed, dc, perl, sort, uniq etc.

Use SSH to log into the docker container (username: root, password: sshpass1), alternatively you can use docker exec command to get into the container.

Log file is in: /opt/local/logs/
